### 應用基本信息

- **完整啟動命令**:
  ```bash
  adb shell am start -n {包名}/{啟動 Activity}
  ```

- **Frida 附加命令**:
  ```bash
  uv run frida -H {設備 IP}:{端口} -l {腳本名稱}.js -F
  # 或使用包名
  frida -U -f {包名} -l {腳本名稱}.js
  ```
